# 재테크 관리 사이트 (GitHub Pages용)

- **오늘 할 일**: 날짜별 할 일 추가/체크(가운데 정렬 + 밑줄 + 취소선)
- **내 자산**: 계좌를 만들고, 계좌 안에 여러 **버킷(용도)** 을 만들어 관리
- **지출/수입 기록**: 기록하면 선택한 버킷 금액이 자동 업데이트
- 데이터는 브라우저(LocalStorage)에 저장됩니다.

## 로컬 실행
- `index.html`을 더블클릭해서 열어도 됩니다.

## GitHub Pages 배포
1) GitHub에서 새 Repository 생성  
2) 이 폴더 안 파일들을 업로드 (index.html / styles.css / app.js)  
3) Repo → Settings → Pages → **Deploy from a branch**  
4) Branch: `main`, Folder: `/ (root)` 선택 → Save  
5) 잠시 후 Pages URL로 접속
